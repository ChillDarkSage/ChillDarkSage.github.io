  屡次安装hexo-renderer-sass都毫无作用，翻了不少博客以后也并没有得出一个有效的解决方案.整个下午我就对着不断生成的这段错误提示发呆……最终在查询的过程中意外发现有人提到dart-sass已经可以无缝替换node-sass，抱着死马当活马医的心态试了试，在CMD中输入：

cnpm install dart.sass

然后：

hexo clean
