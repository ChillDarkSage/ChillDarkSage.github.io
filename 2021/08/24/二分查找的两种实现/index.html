二分查找的两种实现
Aug 24 2021

  二分查找（Binary Search）是在一个有序数组中查找指定键的算法，它的时间复杂度为$O(Log(n))$，具体实现为通过改变$lo$与$hi$两个指针，不断将$mid$所指向的中间键与指定键进行比较；若两键相等，则返回$mid$，否则算法就将查找范围缩小一半。如果指定键小于中间键，则继续在左半边查找；如果指定键大于中间键，就继续在右半边查找，直到找到指定键或者确定指定键不存在。

  下面给出两种实现。

class BinarySearch
{
    public static int Rank1(int[] arr, int key)    //迭代
    {
        int lo = 0, hi = arr.length - 1;
        int mid;
        while (lo <= hi)
        {
            mid = lo + (hi - lo) / 2;
            if (key > arr[mid]) lo = mid + 1;
            else if (key < arr[mid]) hi = mid - 1;
            else return mid;
        }
        
        return -1;
    }
    
    public static int Rank2(int[] arr, int key)    //递归
    {
        private static int Rank2(int[] arr, int key, int lo, int hi)
        {
            if (lo > hi) return -1;
            
            mid = lo + (hi - lo) / 2;
            if (key > arr[mid]) return Rank2(arr, key, mid + 1, hi);
            else if (key < arr[mid]) return Rank2(arr, key, lo, mid - 1);
            else return mid;
        }
        
        return Rank2(arr, key, 0, arr.Length - 1);
    } 
}

在这里，遵循缪欧的法则.
